# Incognito Thoughts

This is a real-time anonymous message board using Firebase Realtime Database.

## How to Use

1. Deploy this folder to GitHub Pages.
2. Open `index.html` in a browser (HTTPS required).
3. Messages will sync in real-time across all devices.

## Features

- Real-time message sync using Firebase Realtime Database
- Input character counter
- Safe HTML sanitization
- Copy shareable link
